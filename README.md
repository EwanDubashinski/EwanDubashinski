Hi 👋 My name is Ivan Dubashinskii
==================================

Full Stack Software Engineer
----------------------------

Embarked on my computer journey at age 5 with Windows 3.11 and classic games 👾, dabbling in QBasic, web development, and Visual Basic 6 during my school years.

Learned the ropes and decided to trade textbooks for real-world experience, landing a gig as a "Programmer" in a bank 🏦. Rode the waves through the 2008 financial crisis, mastering SQL, PL/SQL. 

Made the leap to a banking software vendor, soaking up Oracle DB, SQL, PL/SQL, Java ☕️, and a splash of .NET. 

Decided to spice things up, learned for a bit, then threw myself into the software engineering world. 

Initially, online university disappointment led me to a traditional extramural state university, graduating in 2019. 

Kicked off my Deutsche Telekom (known for T-Mobile brand) journey in 2017 as a Software Engineer, tackling projects from Oracle-based apps to Vue.js and Java Spring Boot. 

Now calling Greece home, but who knows, the journey continues, and maybe it's not my final stop.

*   🌍  I'm based in Thessaloniki, Greece
*   ✉️  You can contact me at dubashinskii at outlook dot com
*   🚀  I'm currently working in [Deutsche Telekom](http://telekom.com/)
*   🧠  I'm learning Designing Data-Intensive Applications
*   ⚡  Considering new opportunities in Portugal and US
